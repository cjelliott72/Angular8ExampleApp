Before running this application in your local development environment, please check you have the required dependencies installed.

Dependency Information

This application was written using Visual Studio 2019 Community Edition version 16.3.7
(download here https://visualstudio.microsoft.com/vs/)

The web client is built using Angular 8 (build 8.3.15) which is included with Visual Studio but requires Node.js version 12.12 to be installed.
(download here https://nodejs.org/en/)

If you have the above version of Visual Studio and Node.js installed, all you need to do is load, build and run the application.

The first time you build the application, Nuget will automatically install the required packages and this can few minutes to complete.

For more information on the project see the 'About' webpage.
